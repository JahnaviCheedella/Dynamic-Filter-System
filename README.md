# Dynamic Filter Component System

A reusable, type-safe dynamic filter system for React applications built with TypeScript and Material-UI.

## Live Demo

**[View Live Demo](https://employee-filter-system.vercel.app)**

## Features

- **7 Different Field Types** - Text, Number, Date, Amount, Single Select, Multi-Select, Boolean
- **Dynamic Operators** - Operators change based on selected field type
- **Real-time Filtering** - Table updates instantly as filters are applied
- **Sortable Table** - Click column headers to sort data
- **50+ Sample Records** - Realistic employee data for testing
- **Type-Safe** - Full TypeScript implementation with strict typing
- **Modular Architecture** - Reusable, well-organized components
- **Material-UI Design** - Professional, accessible UI components
- **Nested Object Support** - Filter on nested properties (e.g., address.city)
- **Array Filtering** - Handle multi-value fields (e.g., skills)

## Prerequisites

- Node.js 16+ and npm/yarn
- Basic knowledge of React and TypeScript

## Installation

1. **Create project directory and navigate to it**

```bash
mkdir dynamic-filter-system
cd dynamic-filter-system
```

2. **Copy all the provided files into the project structure**

3. **Install dependencies**

```bash
npm install
```

4. **Run the development server**

```bash
npm run dev
```
